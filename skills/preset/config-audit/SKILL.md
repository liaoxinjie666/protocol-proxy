# 配置审计

请按以下步骤审计系统配置：

1. 调用 `get_providers` 获取所有供应商配置
2. 调用 `get_proxies` 获取所有代理配置
3. 调用 `get_settings` 获取系统设置
4. 调用 `get_key_health` 检查密钥状态
5. 检查配置文件内容（如有需要）

审计清单：

### 安全检查
- [ ] 是否有未配置认证的代理？
- [ ] API Key 是否有硬编码泄露风险？
- [ ] 是否有使用弱认证令牌的代理？

### 配置完整性
- [ ] 所有供应商是否配置了 API Key？
- [ ] 代理是否都有有效的供应商引用？
- [ ] 是否有孤立的供应商（未被任何代理使用）？

### 最佳实践
- [ ] 是否配置了密钥轮换？
- [ ] 路由策略是否合理？
- [ ] 是否有备用供应商配置？
