# 性能报告

请按以下步骤生成性能报告：

1. 调用 `get_usage_stats` 获取 Token 使用统计
2. 调用 `get_recent_requests` 获取最近请求记录
3. 调用 `get_system_status` 获取系统运行时长
4. 分析请求分布和性能指标

输出报告：

## 性能报告

### 请求概览
- 总请求数: X
- 成功率: X%
- 平均延迟: Xms

### Token 使用
- 总输入 Token: X
- 总输出 Token: X
- 按模型分布: 列出各模型使用量

### 延迟分析
- P50: Xms
- P95: Xms
- P99: Xms
- 最慢请求: 列出详情

- 基于数据给出性能优化建议
