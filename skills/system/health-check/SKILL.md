# 系统健康检查

请按以下步骤执行系统健康检查：

1. 调用 `get_system_status` 获取整体概览
2. 调用 `get_key_health` 检查 API 密钥健康状态
3. 调用 `get_proxies` 检查所有代理运行状态
4. 调用 `get_providers` 检查供应商配置完整性

然后输出健康报告，格式：

## 系统健康报告

**整体状态**: 健康/警告/异常

### 代理状态
- 运行中: X 个
- 已停止: X 个
- 异常: 列出异常代理

### 密钥健康
- 正常: X 个
- 限流中: X 个
- 失效: 列出失效密钥

### 建议
- 列出需要关注的问题和优化建议
