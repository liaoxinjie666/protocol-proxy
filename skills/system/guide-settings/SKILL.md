# 系统设置指南

## 一、配置管理

配置管理用于备份、迁移和恢复系统配置。

### 配置项详解

| 配置项 | 含义 | 说明 |
|--------|------|------|
| **导出配置** | 将所有配置保存为 JSON 文件 | 不包含 API Key，仅导出配置结构 |
| **导入配置** | 从 JSON 文件恢复配置 | 支持合并或覆盖模式 |
| **版本历史** | 查看配置变更快照 | 支持回滚到历史版本 |

### 导出配置

导出的 JSON 文件包含：
- 供应商配置（不包含密钥）
- 代理配置
- MCP 服务配置

可用于：
- 配置备份
- 迁移到新服务器
- 分享配置（不含敏感信息）

### 导入配置

| 模式 | 行为 | 适用场景 |
|------|------|----------|
| **合并** | 按 ID 去重，新增导入项，同 ID 覆盖 | 补充配置、增量迁移 |
| **覆盖** | 完全替换现有配置 | 全新部署、完全迁移 |

导入前会显示预览：
- 将导入的供应商数量
- 将导入的代理数量

### 版本历史

系统会保存配置变更历史：
- 记录每次保存的配置快照
- 显示保存时间和变更说明
- 支持回滚到任意历史版本

### 回滚操作

1. 打开「设置」→「配置管理」→「版本历史」
2. 选择要回滚的版本
3. 点击「回滚」按钮
4. 确认回滚操作

回滚会恢复该版本的所有配置（供应商、代理、MCP）。

## 二、主题设置

主题设置用于自定义界面的外观配色。

### 主题配置

| 主题 | 特点 | 适用场景 |
|------|------|----------|
| **深色**（默认） | 深色背景，浅色文字 | 夜间使用、减少眼睛疲劳 |
| **浅色** | 浅色背景，深色文字 | 白天使用、高对比度需求 |
| **午夜紫** | 紫色调的深色主题 | 个性化、创意工作 |
| **森林绿** | 绿色调的深色主题 | 放松、自然感 |
| **日落橙** | 橙色调的深色主题 | 温暖、活力感 |
| **海洋青** | 青色调的深色主题 | 科技感、专业感 |
| **樱花粉** | 粉色调的深色主题 | 甜美、柔和感 |

### 切换主题

| 方式 | 操作 |
|------|------|
| **设置页面** | 「设置」→「外观」→「主题」下拉框选择 |
| **快捷切换** | 点击侧边栏底部的月亮图标切换深色/浅色 |

切换立即生效，无需保存。
