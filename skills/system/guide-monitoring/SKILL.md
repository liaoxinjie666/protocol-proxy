# 监控与日志指南

## 一、用量统计

用量统计帮助追踪 API 使用情况，包括 Token 消耗和请求量。

### 配置项详解

| 配置项 | 含义 | 说明 |
|--------|------|------|
| **时间范围** | 统计的时间维度 | `每日`：按天统计<br>`每月`：按月统计<br>`每年`：按年统计 |
| **代理筛选** | 只统计特定代理的数据 | 选择特定代理或「全部代理」 |
| **日期范围** | 自定义统计的时间段 | 设定开始和结束日期 |

### 统计数据说明

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| **总 Token** | 统计周期内的总消耗 | 输入 + 输出 token 之和 |
| **输入 Token** | 发送给模型的 token 数量 | 请求中的 prompt token |
| **输出 Token** | 模型返回的 token 数量 | 响应中的 completion token |
| **请求数** | API 请求的次数 | 成功 + 失败的总请求数 |
| **按模型统计** | 每个模型的使用量明细 | 按模型分组显示详细数据 |

### 导出数据

点击「**导出 CSV**」按钮下载统计数据：
- 包含时间、模型、Token 数量等列
- 可在 Excel 中进一步分析

### 筛选条件使用

1. 选择时间范围（每日/每月/每年）
2. 选择代理（全部或特定代理）
3. 设置日期范围（可选）
4. 系统自动更新统计数据

### 估算值说明

部分统计可能包含估算值（显示「含估算值」提示）：
- 当 API 响应不包含完整 token 计数时
- 使用请求模型的最大 token 数进行估算
- 估算值会标记以便识别

## 二、请求日志

请求日志实时记录所有通过代理的 API 请求。

### 配置项详解

| 配置项 | 含义 | 说明 |
|--------|------|------|
| **代理筛选** | 只显示特定代理的请求 | 选择特定代理或「全部代理」 |
| **状态筛选** | 只显示特定状态的请求 | `全部状态`、`成功`、`失败`、`429 限流` |
| **模型过滤** | 搜索框过滤特定模型 | 输入模型名部分文字搜索 |

### 日志内容说明

每条请求日志包含以下字段：

| 字段 | 含义 | 示例 |
|------|------|------|
| **时间** | 请求发生的时间 | `2024-01-15 14:30:25` |
| **代理** | 接收请求的代理名称 | `OpenAI 代理` |
| **协议** | 请求使用的协议格式 | `OpenAI`、`Anthropic` |
| **模型** | 请求的 AI 模型 | `gpt-4`、`claude-3-opus` |
| **状态** | 请求结果 | `成功`、`失败`、`429` |
| **Tokens** | 本次请求消耗的 token 数 | `1250` |
| **延迟** | 请求处理时间（毫秒） | `850ms` |
| **供应商** | 实际转发的供应商 | `OpenAI 官方` |
| **Key** | 使用的 API Key（部分隐藏） | `sk-****...xxxx` |

### 状态说明

| 状态 | 含义 | 处理建议 |
|------|------|----------|
| **成功** | 请求正常完成 | 无需操作 |
| **失败** | 请求出错（服务器错误、超时等） | 检查供应商状态和网络 |
| **429** | 触发限流 | 增加 Key 或降低请求频率 |

### 实时更新

请求日志页面使用 WebSocket 实时更新：
- 新请求自动追加到列表顶部
- 状态变更实时反映
- 连接状态显示在工具栏

### 导出和清空

| 操作 | 说明 |
|------|------|
| **导出** | 下载日志为 CSV 文件，包含所有字段 |
| **清空** | 清除当前显示的日志记录 |

## 三、系统日志

系统日志记录系统运行时的各种事件和错误。

### 日志配置

| 配置项 | 含义 | 选项 |
|--------|------|------|
| **显示行数** | 加载的日志条数 | 100、200、500、1000 |
| **刷新** | 重新加载最新日志 | 点击按钮刷新 |

### 日志级别说明

| 级别 | 含义 | 触发场景 |
|------|------|----------|
| **INFO** | 一般信息 | 代理启动/停止、请求处理完成 |
| **WARN** | 警告信息 | 密钥限流、配置过期 |
| **ERROR** | 错误信息 | 连接失败、认证失败、异常捕获 |
